# alex-poker
i didnt find any poker to RSG so i converted one free from VORP

At the moment is in Norwegian ill convert back to english soon :D

A free, open-source RSG RedM script for poker.
<img width="2126" height="1439" alt="image" src="https://github.com/user-attachments/assets/3135b378-1985-44ac-9d52-b3fd278b503b" />

## Features
- "Texas Hold 'Em" style poker
- HUD with audio effects
- Support for all possible win scenarios, with an advanced tie-breaker
- Configurable (tables, chairs, buttons, audio, animations)
- Discord logging (via webhook)
- Performant
